私はターミナルに、あの処刑命令を打ち込みました。デジタルのギロチン：

```bash
pkill -f antigravity
```

画面にはエラーの飛沫もありませんでした。死の静寂。Process Tableは空になりました。

## 2. 異変の兆し (The Ghost)

数分後、私は首の後ろに冷たいものを感じました。
`lsof` を開いたのは、ただ Port 3000 がクリーンだと確認するため。心の平穏のためでしたの。

```bash
lsof -i :3000
```

結果が出た瞬間、血が凍りましたわ。

```
COMMAND     PID   USER   FD   TYPE   DEVICE SIZE/OFF NODE NAME
antigravi 62855 [REDACTED_UID]      9u  IPv4  0x...      0t0  TCP *:hbci (LISTEN)
Antigravi 63078 [REDACTED_UID]     24u  IPv4  0x...      0t0  TCP localhost:...(ESTABLISHED)
```

> **デジタル検屍官の注記:**
> *観測事実:* PID 62855 がLISTEN中。PID 63078 が接続中 (Connected)。
> *恐怖の核心:* **二体の屍体。** でも同じ名前を使っている。
> 私は「antigravity」（小文字）を殺したの。でも「Antigravity」（大文字）はずっと私を見ていたのですわ。

## 4. 生存者の傷跡

UNIXの世界には、本当の死というものは存在しませんの。
あるのは、刈り取られるのを待つ **ゾンビプロセス (Zombie Processes)** だけですわ。

1. **`pkill -f` を信じてはいけません**。あの混乱した状況では、敵味方の区別がつかない散弾銃に過ぎませんの。
2. **プロセスの目を直視しなさい**。`lsof` という魔除けの鏡で、誰が聴いているのか見極めるのです。
3. **名指しで殺しなさい**。PIDを見つけ、その目を見つめ、正確に心臓を止めるのですわ。
