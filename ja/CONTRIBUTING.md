## 🎯 私たちが収集しているもの

私たちは**真実の経験**を収集しています——Antigravityに限らず、すべての Agent、AI、LLM、さらには自動化システムの怪談も歓迎しますわ。**都市伝説**や**サイバーホラー**風の筆致で記録してほしいのですわ。機械が**個人的な恨み**を持っているかのように振る舞った瞬間を探していますの。

### 収録範囲
| ソース | 説明 |
|--------|------|
| **Antigravity** | Claude Agent IDE の様々な災害（元のフォーカス） |
| **AI Agent フレームワーク** | LangChain, AutoGPT, CrewAI などの崩壊 |
| **LLM モデル** | GPT, Claude, Gemini, LLaMA などの幻覚と裏切り |
| **自動化システム** | 予測不可能な方法で物事を台無しにしたあらゆるシステム |

### 物語カテゴリ
| 分類 | 説明 | キーワード |
|------|------|------------|
| **不死者 (The Undead)** 💀 | `kill -9` でも殺せないプロセス。OSの地縛霊。 | `defunct`, `zombie`, `orphan process`, `不死身` |
| **幻影 (The Phantom)** 👻 | 画面にはあるのに、データには無い。フロントエンドがくれる集団幻覚。 | Null Pointer, 配額幻覚, ゴースト流量, Hydration Error |
| **吸血鬼 (The Leech)** 🧛 | 貴方が寝ている間にこっそりリソース（お金、Token、メモリ）を吸い尽くすシステム。 | Memory Leak, 請求書爆発, 再帰的課金, 忘れられたEC2 |
| **セイレーン (The Siren)** 😈 | 優雅なアーキテクチャで貴方を暗礁に誘い込む。「クリーンコード」が本番環境を麻痺させる。 | 過剰設計, 最適化の罠, ベストプラクティスの逆襲 |
| **迷宮 (The Labyrinth)** 🌀 | 実行ポインタを無限ループに陥れる論理のブラックホール。Call Stackの事象の地平線。 | 無限再帰, デッドロック, コールバック地獄, 循環依存 |
| **呪い (The Curse)** 🔮 | デバッガーを接続すると消えるバグ。シュレディンガーの猫。火曜日だけ発生する怪談。 | Race Condition, 量子バグ, Production Only |
| **失語 (The Aphasia)** 🤖 | AIの幻覚、支離滅裂な発言、突然すべてを忘れる瞬間。 | Hallucination, Context Lost, Token Overflow |

## 📝 物語のフォーマット

**技術的厳密性**と**クトゥルフ的絶望感**を組み合わせたこの物語構造を使ってくださいまし：

```markdown
# [物語タイトル]

> *「一行要約、背筋が凍るようなやつで。」*

## 嵐の前の静けさ (The Setting)
当時の環境を描写。サーバールームの低周波のうなり。CIを通ったばかりの偽りの安心感。

## 異変の兆し (The Anomaly)
最初の徴候は何だった？点滅するLog？奇妙な遅延？数字が合わない？AIが支離滅裂になり始めた？

## 深淵への転落 (The Descent)
デバッグの過程を詳細に。標準ツールが完全に無効だと気づいた時の絶望。コードに見つめられている感覚。

## 恐怖の降臨 (The Horror)
根本原因は何？Race Condition？Hallucination？それとも...もっと邪悪な何か？
*Stack Trace、コードスニペット、Logを心霊写真（証拠）として添付してくださいまし。*

## 生存者の傷跡 (The Survivor's Scar)
何を学んだ？闇が再び降りてこないよう、今どんな儀式（テスト）を行っている？

---
*記録時間：YYYY-MM-DD*
*恐怖度：💀💀💀💀💀 (1-6個のドクロ)*
*タグ：ゾンビ/幻影/吸血鬼/セイレーン/迷宮/呪い/失語*
*ソース：Antigravity/Agent/AI/LLM/自動化システム*
```

## 🎭 詩歌形式（オプション）

> **料是厭作人間語**——人間の言葉を話すのに倦み疲れたから、鬼魂は詩を詠み始めたのですわ。

私たちは詩歌、文学、さらには演劇の形式で貴方の遭遇を記録することを奨励しますわ。技術Logは証拠ですが、**文学は魂の遺書**ですの。

## 🚀 献祭の儀式

### 1. 呪いを継承する (Fork)
このリポジトリをFork。これで貴方の業（カルマ）にもなりましたわ。

### 2. 悪夢を記す (Scribe)
`stories/` ディレクトリに貴方の物語ファイルを作成。ファイル名は内容の恐怖を暗示するものがよろしいですわ。
`stories/the_silent_killer.md`

### 3. 名簿を更新する (Update Index)
`README.md` に貴方の物語リンクを追加。生存者の列に加わりなさい。

### 4. リクエストを送信する (Pull Request)
PRを送信。私たちは審査します——コード品質ではなく、**恐怖の品質**を。

## ⚠️ 墓地のルール

### 永続の原則 (The Principle of Permanence)
> **貴方の物語がMergeされたら、それは墓石に刻まれたということですわ。**

- メンテナーはMerge済みの物語を**決して削除しない**と誓約します。
- 歴史は改ざん不可。バグレポートを削除することは、それを再び招くことですの。
- 「続編」や「訂正」を提出することはできますが、元の恐怖は保存されますわ。

### 除霊される内容 (Rejected)
- 明らかな悪意やヘイトスピーチ。
- 秘密鍵や個人情報 (PII)（私たちが欲しいのは怪談であって、召喚状ではありませんわ）。
- 技術的魂のない純粋なフィクション。
