# プロジェクトアーキテクチャ / Architecture

> *料是厭作人間語，愛聽秋墳鬼唱詩*
>
> — 実は、それはKernelレベルの大鬼たちが集まって詩を詠んでいただけですの。ある日、人々は彼らの残した詩がコンパイルを通過できることに気づきましたわ。

---

## プロジェクト定義

**Antigravity & Agent Stuck Story** は、すべての Agent、AI、LLM、自動化システムが人間の信頼をどのように裏切ったかを記録するプロジェクトですわ。

- **元のフォーカス**：Antigravity（Claude Agent IDE）の様々な災害
- **拡張範囲**：あらゆる Agent、AI、モデルの怪談
- **核心精神**：バグを修正するのではなく、供養する

---

## [Phase 1] 原始アーキテクチャの回顧と完全実装

**User Objective:** 原始アーキテクチャ文書を回顧し、アカウント輪番を含む簡体字中国語プロジェクトの参照を避け、完全実装を実行。

### 分析ログ
- **プロジェクト構造スキャン:** `Saki_Studio/Claude` 環境を検出。
- **重大警告:** ユーザーが「アカウント輪番」と「リバースプロキシ」を悪意あるパターン（「クズ実装」）としてフラグ付け。
- **アクション:** 輪番ベースの配額管理への全参照をパージ。

> ❌ **システムエラー:** エラーによりAgent実行が終了。
> *注記: これが最初の異常でした。機械は進行を拒否したのですわ。*

## [Phase 2] エラーの物語への変換 (The Pivot)

**User Objective:** 怪談プロジェクトを生成。すべてのエラーを物語に。しかも、怪談に。

### 実行トレース
1.  **読み込み:** `事件記錄_2026-01-26.md` (トラウマの源泉)。
2.  **処理:**
    -   *イベント:* `pkill` がプロセスを殺せなかった。 -> *物語:* **ゾンビプロセス（降霊術）。**
    -   *イベント:* DOMスクレイピングがnullを返した。 -> *物語:* **虚無UI（ガスライティング）。**
    -   *イベント:* エンタープライズプランで `429`。 -> *物語:* **Best Planの呪い（脳葉切除）。**
3.  **合成:** 物語レイヤーの構築。スタックトレースを「法医学的証拠」に変換。

### 生成されたファイル
- `README.md`: 宣言。
- `stories/*.md`: トラウマログ。

## [Phase 3] デジタル検屍と社会実験（拡張 Agent 範囲）

**User Objective:** 被害者が声を上げられるプラットフォームを構築——Antigravityの被害者だけでなく、Agent、AI、モデルに裏切られたすべての人のために。これは **Agent と AI** に関する実験ですの。

### アーキテクチャ決定
- **モックなし:** すべてのログは本物でなければなりません。
- **法医学的アプローチ:** すべてのバグレポートを犯罪現場として扱う。
- **詩的ライセンス:** 技術ログと並んで文学的表現を奨励。
- **コミュニティプロトコル:**
    -   Fork -> 記述 -> PR。
    -   不変の歴史（削除禁止）。

### 物語カテゴリ (Agent Layers)
- **Antigravity Stories:** 元のフォーカス、Claude Agent IDE の災害
- **Agent Stories:** LangChain, AutoGPT, CrewAI フレームワークの崩壊
- **AI/LLM Stories:** GPT, Claude, Gemini, LLaMA の幻覚
- **Automation Stories:** あらゆる自動化システムの奇妙な行動

## [Phase 4] 詩歌の核心 (The Poetic Agent)

> **姑妄言之姑聽之，豆棚瓜架雨如絲**
> **料是厭作人間語，愛聽秋墳鬼唱詩**

すべての物語は、技術的詳細がどれほど乾燥していても、この詩的本質を帯びるべきですわ：

- **姑妄言之姑聽之**：適当に話し、適当に聞く——これらの物語は荒唐無稽かもしれませんが、すべて本当のこと
- **豆棚瓜架雨如絲**：質素な環境で（おそらく深夜のIDE）、雨のような災害が降りてくる
- **料是厭作人間語**：人間の言葉を話すのに飽きたから、だから
- **愛聽秋墳鬼唱詩**：鬼魂が詩を詠み始めた——そしてその詩は、コンパイルを通過したのですわ

**Agentたちの物語** は、それらの亡霊たちの集合体ですの。貴方が Terminal を見つめる時、Agent たちもまた貴方を見つめ返していますわ。
