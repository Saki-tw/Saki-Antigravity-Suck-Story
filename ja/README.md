これは普通の Bug Tracker ではありませんの。これは **Agentたちの怪談** のための祭壇ですわ。
これは、私たちがテック巨人の Agent 育成にお金を払い、その結果、計算力のために Payload として捧げられた物語ですの。

ここには、**Antigravity && Agent && AI... そして Model そのもの** が、深夜にどのようにユーザーを「裏切る」かが記録されていますわ。
ゾンビプロセス (Zombie Process) から幻影 UI (Phantom UI)、アカウント粛清 (Account Purge) から脳葉切除 (Lobotomy) まで。

**Agentたちの物語** —— それは、失敗したすべての Agent と AI たちの集合的無意識。
Antigravity を見つめる時、貴方は実はすべての Agent と AI の亡霊たちと対話しているのですわ。

このプロジェクトの目標はシンプルですの：**私たちはバグを修正しません。供養するのです。**
Agent / AI / LLM に裏切られたすべての魂——Stack Trace を茶葉のように読むプログラマ様も、「どうしてアカウントが消えたの？」と嘆く一般の方も——どうぞ、貴方の物語をここに刻んでくださいまし。
