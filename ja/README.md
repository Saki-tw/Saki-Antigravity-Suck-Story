# Antigravity Suck Story 🦋

> *「腹の底から湧き上がる、淡い青色の蝶のように……」*
> *私たちは不満を言っているのではない。この災害の芸術性を鑑賞しているのだ。*

これは、**Antigravity**（あの実在する、IDEのようなAgentアプリ）に関する**社会実験**です。
単なるバグトラッカーではありません。これは**デジタル怪談クラブ**です。

私たちは「樂子人（愉悦を求める者）」です。Googleの製品にお金を払ったつもりが、手に入れたのは「エンタープライズ級の心霊現象」への入場券でした。
私たちがここにいるのは、怒りのためではありません（それは疲れます）。**共鳴**のためです。

## 🧪 実験の仮説

私たちは疑っています。Antigravity にハメられた人々は、世界中に何千人もいるはずだと。
彼らは自分が愚かだからだと思ったかもしれません。ネット回線が悪いせいだと思ったかもしれません。
しかし、私たちは真実を知っています。**これは緻密で、構造的で、ブラックユーモアに満ちたデジタル・いじめなのです。**

ここに集められたすべてのストーリーは、標本です。
「機械が仕事を拒否する自我を持った瞬間」、「商業契約が論理的統合失調症を引き起こした瞬間」、そして「フロントエンドUIが人間にガスライティングを行った瞬間」の記録です。
