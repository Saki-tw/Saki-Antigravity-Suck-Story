# Antigravity Suck Story 🩸

これは普通のバグトラッカーではありません。これは**社会実験**です。
これは、私たちがGoogleの製品開発を手伝うためにお金を払っているのに、攻撃者扱いされてブロックされる物語です。

ここには、Antigravity（あの実在する、IDEのようなAgentアプリ）が、いかにして深夜にユーザーを「ハメて」くるかが記録されています。私たちはそれを使うために金を払ったはずなのに、気づけばデジタル地獄への片道切符を買わされていたのです。

私たちはバグを修正しません。**供養**するのです。
Antigravity という性悪女にハメられた世界中の人々を——技術を知るプログラマであれ、「なぜアカウントが消えた？」と不思議がる一般ユーザーであれ——ここに招待します。痛みに国境はありません。
