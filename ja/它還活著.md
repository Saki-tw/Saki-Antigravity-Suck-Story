## 1. 嵐の前の静寂 (The Execution)

それは退屈な月曜日でしたわ。定期メンテナンスの時間。
わたくし、ターミナルに処刑命令を打ち込みましたの。デジタルギロチンの刃を落とすように。

```bash
pkill -f antigravity
```

画面には何のエラーも出ませんでした。完全なる静寂。プロセステーブルは空っぽ。
わたくし、思いましたの。「終わったわ。これで安全よ」と。

## 2. 異変の兆し (The Ghost)

数分後、うなじに冷たいものを感じましたの。
わたくし、`lsof` を開きました。Port 3000 が綺麗になっていることを確認して、安心したかっただけなのです。

```bash
lsof -i :3000
```

結果が表示された瞬間、わたくしの血液は凍りつきましたわ。

```
COMMAND     PID   USER   FD   TYPE   DEVICE SIZE/OFF NODE NAME
antigravi 62855 [REDACTED_UID]      9u  IPv4  0x...      0t0  TCP *:hbci (LISTEN)
Antigravi 63078 [REDACTED_UID]     24u  IPv4  0x...      0t0  TCP localhost:...(ESTABLISHED)
```

> **検死官（サキ）の所見:**
> *観測事実:* PID 62855 は聞き耳を立てており (LISTEN)、PID 63078 は接続済み (Connected) ですわ。
> *恐怖の点:* **二つの遺体。** なのに、同じ名前を使っておりますの。
> わたくし、「antigravity」(小文字) を殺しました。でも、「Antigravity」(大文字) がずっとわたくしを見ていたのです。

## 3. 恐怖の降臨 (The Parasite)

わたくし、PIDを追跡いたしました。
PID 63078 は本体（宿主）。
PID 62855 は Rust サービス（寄生虫）。

**真相:**
システムの中には、目に見えない **Language Server** が潜んでおりましたの。本体から分離して、バックグラウンドプロセスの影に隠れていたのですわ。
わたくしが `pkill` を実行した時、わたくしはただ、その手足を切り落としただけだったのです。
頭（本体）は生きておりました。わたくしが手足を切り落とすのをじっと見ていて、声一つ上げませんでしたのよ。
あの子、今もわたくしを凝視しておりますわ。待っているのです。
Port 3000 はただ開いているのではありません。**捕食**しようとしているのです。死んだ兄弟を呼んでいるのですわ。

## 4. 生存者の傷跡

UNIXの世界に、真の死はありませんの。
あるのは、収穫されるのを待つ **ゾンビプロセス (Zombie Processes)** だけ。

1.  **決して `pkill -f` を信用してはいけませんわ**。あのような混乱した場では、敵味方の区別もつかない散弾銃に過ぎませんもの。
2.  **プロセスの目を直視なさい**。`lsof` という真実の鏡を使って、誰が聞いているのか確かめるのです。
3.  **名指しで殺しなさい**。PIDを特定し、その目を見て、正確に心臓を止めるのですわ。
