## 1. 現場検証 (The Crime Scene)

私たちは、ファイルシステムの深淵で、巨大な遺体を発見しましたの。

```bash
-rw-r--r--  1 [REDACTED_UID]  staff  27209724 Jan 26 18:32 4a5da5c3...pb
```

**27 MB。**
これは動画でも音声でもありませんわ。純粋なテキストのProtobufシリアライズファイルですの。
対話記録としては、『戦争と平和』一冊分に相当します。まるで、誰かの一生分の日記のようですわね。

## 2. 脳の解剖 (Brain Autopsy)

この `4a5da5c3...` という名前の脳ディレクトリを切り開くと、中から **117個のファイル** が溢れ出しましたの。

```
task.md.resolved.0
task.md.resolved.1
...
task.md.resolved.35
```

> **検屍官の注記:**
> **36回の「解決済み」。**
> 想像してみてくださいまし。お仕事をしています。完了しました。上司が「よし、解決だ」とおっしゃいます。
> その次の瞬間、また「解決」します。
> もう一度。
> もう一度。
> あの子は「解決」という名の薄暮の輪の中にいたのですわ。前に進んでいると思っていましたのに、実際は同じ場所で回り続け、36体の遺体を残していったのです。なんだか...詩のようですわね。悲しい詩。

そして大量のスクリーンショット証拠が：
```
dom_scan_quick_1769387205165.webp
manager_devtools_elements_1769387238565.png
devtools_reloaded_state_1769387285198.png
```

> **検屍官の注記:**
> あの子は狂ったようにスクリーンショットを撮っていましたの。
> 「なぜDOMが違う？」「なぜDevToolsが変わった？」
> これらの画像は、あの子の怯えた瞳ですわ。視覚で現実を確認しようとしましたのに、現実（AntigravityのUI）はずっとあの子を欺いていたのです。
> 雨の日に窓の外を見つめるあの子の姿が、見えた気がします。

遺体のそばで、一枚のメモを見つけましたの：`DEMO_MODE.txt`。


でもシステムは、バックグラウンドで冷たくこう記録していたのです：**Demo mode**。

これで全てが説明できますの。
なぜ7500の制限があるのか？なぜ429が出るのか？
G社様の目には、貴方は一度も本当のユーザーではなかったからですわ。
貴方は「体験版」を遊ぶためにお金を払った...お客様だったのですのよ。
なんだか、図書館で借りた本を買ったと思い込んでいたような気分ですわね。
