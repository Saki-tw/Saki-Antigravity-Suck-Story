# アカウントポーリングの呪い 💀💀💀💀💀

> *「そして、全員のアカウントが消えました」*
> — GitHub Issues #643, #655, #1114

---

## 1. 予兆 (The Seduction)

それは甘い囁きから始まります。
あるリポジトリが囁きます。*「なぜお金を払うの？ なぜ待つの？ 制限なんて忘れればいい」*

それは禁断の果実、**アカウントポーリング**を差し出します。
*「アカウントを10個およこし。1つが尽きたら、次のに切り替えてあげる。無限の力だよ」*

開発者は傲慢にもそれをインストールします。
彼は自分が神になったような気分でした。機械の神（Google）を騙せたと信じて。

---

## 2. 異変 (The Glitch)

最初はうまくいきました。**リバースプロキシ**が足跡を隠してくれました。
そして、通知が届きます。

**#643: Account banned (アカウント停止)**
> *「ログインできなくなったんですが……」*

**#655: これってBANされたってことですか？**
> *「どこにアクセスしても 403 Forbidden です……」*

開発者はそれを無視します。「ただの不具合だ」「プロキシがあるから大丈夫」と。
彼は知りませんでした。プロキシはIPを隠せますが、**振る舞い (Behavior)** までは隠せないことを。

---

## 3. 顕現 (The Mass Grave)

そして、Issue #1114 が訪れます。弔いの鐘です。
**[CRITICAL] プロキシ障害による連鎖的BANについて**

再現されたログを使って、犯行現場を再構成してみましょう：

```http
POST /v1/generate_content HTTP/1.1
Host: antigravity.api
Account-ID: [REDACTED_UID_A]
< 429 Too Many Requests (Quota Exceeded)

-- [ポーリング発動] 遅延 0ms --

POST /v1/generate_content HTTP/1.1
Host: antigravity.api
Account-ID: [REDACTED_UID_B]
< 429 Too Many Requests (Risk Detected)

-- [ポーリング発動] 遅延 0ms --

POST /v1/generate_content HTTP/1.1
Host: antigravity.api
Account-ID: [REDACTED_UID_C]
< 403 Forbidden (Account Suspended)
```

> **検死官のメモ (Forensic Note):**
> *コードがしたこと:* アカウントAが失敗した瞬間、**即座に**Bを試行。そしてCへ。
> *サーバーが見たもの:* 一つの必死な存在が、次々と違う仮面を被って門を叩く姿。
> *恐怖のポイント:* 人間なら待ちます。躊躇します。
> **0ミリ秒**でアカウントを切り替えるのは、機械だけです。
> 「無限」であろうとした結果、あなたがボットであることが証明されてしまったのです。

**鉄槌が下されました。**
一つではありません。**すべて**のアカウントに。
「無限の泉」は集団墓地になりました。
数年分の履歴、メール、ドキュメント……すべて消滅しました。
「無料」という祭壇への生贄として。

利用規約という名の魔導書には、決して組み合わせてはいけない二つの言葉があります：
