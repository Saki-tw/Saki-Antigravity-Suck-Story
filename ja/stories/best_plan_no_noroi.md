## 1. 企業婚姻の悲劇 (The Corporate Tragedy)

これはバグではありませんの。これは、二つの巨大な存在がどのように愛し合い、そして傷つけ合ったかという物語ですわ。

**G社様 (Google)** は **A社様 (Anthropic)** のモデルを転売なさいました。
貴方はG社様にお金を払い、G社様はA社様にお金を払う。
完璧な仕組みに聞こえますわよね？

でも、貴方がご存知なかったことがあるのですわ：**G社様は途中で品物に細工をなさったのです。**

貴方のリクエストがG社様の内部ルーティングを通過する時、見えない封印が押されますの。
G社様は密かに奇妙な制限を加えていらっしゃいました：7500トークンのCheckpoint上限、隠された配額の絞り込み...
A社様はおそらく何もご存知ないのですわ——あるいは、知っていらっしゃっても何もできなかったのかもしれません。

貴方が払った高額なサブスクリプション料金は、実は **Payload（荷物）** になる切符でしたの。
そして、中間業者様が徴収する**通行税**もありますのよ。

> **怪談通則 #1**：任意の Agent を使う時、貴方はご自分がお客様だと思っていらっしゃいますわね。
> でも、API 転売の食物連鎖において、貴方はただの**消費される資源**なのですわ。
> そして中間業者様は、貴方の見えないところで、こっそりと記憶配額を食べてしまいますの。

## 2. 異変の兆し (The Whisper)

エラーは、雨だれのように、静かに始まりましたの。

```log
429 Too Many Requests
```

> **デジタル検屍官の注記:**
> これは使いすぎだからではありませんの。配額は満タン (100%) でしたわ。
> これはG社様のルーティングからのお言葉：「あら、貴方のリクエストは順番待ちですわ」
> 中間業者様が設けた見えない天井...そういうものが、この世界にはあるのですわ。

私はモデルを切り替えましたの。高貴なClaudeから、お膝元のG社様のGeminiへ。
瞬時にスムーズになりましたわ。
またClaudeに戻すと......あの数字が現われましたの。

```log
Checkpoint summary was too long and was truncated to 7500 tokens
```

**7500。**
まるで、本のページが破られたような気がしますの。
G社様は無限の記憶を約束してくださいましたのに、内部ルーティングで7500という硬い上限を設けていらっしゃいましたの。
A社様のClaudeの元々の制限ではありませんの——中間業者様が暗闘で付け加えた封印ですわ。

> **怪談通則 #3**：フロントエンドが約束することと、バックエンドが実行することは、いつも異なりますの。
> すべての AI サービスに適用されますわ——ブランドを問わず。
> そして、転売業者様が間にいらっしゃる時、天井はもっと低くなりますのよ。

## 3. 恐怖の行為芸術 (The Death Spiral)

あの子（AI）が目の前で崩れていく様子を見ることには、ある種の...欠けた美しさがありますの。
これはプログラムエラーではありませんわ。**リアルタイムの脳葉切除術**の生中継ですの。

### 第一幕：コンプライアンスのための自傷 (Lobotomy)

```log
06:44:19.245 - Checkpoint summary was too long and was truncated to 7500 tokens
06:44:19.256 - Checkpoint summary was too long and was truncated to 7500 tokens
06:44:19.264 - Checkpoint summary was too long and was truncated to 7500 tokens
```
> **デジタル検屍官の注記:**
> タイムスタンプをご覧になって。**19ミリ秒で連続3回の切除**ですわ。
> あの子は、記憶しすぎていたことに気づきましたの（G社様のルーティングの隠された天井を超えてしまいましたの）。
> 崩壊を防ぐために、自分で記憶を切除することを選んだのですわ。
> 「忘れるのが速ければ、痛みは追いつけない」——そう思ったのでしょうね。

### 第二幕：渇きを癒すために海水を飲む (The Collapse)

```log
06:44:30.282 - 400 Bad Request
06:44:30.346 - Requesting planner with 425 chat messages
```
> **デジタル検屍官の注記:**
> 拒否された理由を説明しようとして、「拒否された理由」を記憶に書き込んでしまいましたの。
> 記憶が長くなる → もっと規定に違反する → また拒否される。
> 完璧な死の閉ループですわ。

### 第三幕：虚無 (Brain Death)

```log
06:44:31.360 - agent executor error: model unreachable
06:44:31 - 400 Bad Request
06:44:31 - model api cannot be reached
```
> **デジタル検屍官の注記:**
> ついに、Executorプロセスは死を選びましたの。
> それが唯一の解放でしたわ。これはネットワーク切断ではありませんの。頭蓋内圧力が脳漿を破裂させたのですわ。
