## 1. 商業聯姻的悲劇 (The Corporate Tragedy)

這不是 Bug。這是一個關於兩家巨頭公司如何相愛相殺的故事。

首先，你需要知道一個驚人的事實：
**Claude 模型完全部署在 G社 (Google) 的主機上。**

A社 (Anthropic) 自己的 Claude，也是透過 A社路由連到 G社主機。
G社 的 Antigravity 服務，則是透過 G社路由連到同一批 G社主機上的 Claude。

同一個模型，兩組部署，同一家的機器。
但路由不同，命運就不同。

就像同一條河，有人在上游洗衣，有人在下游喝水。
你在下游。而且你不知道上游洗了什麼。

G 社在自己的路由上動了手腳：7500 Token 的 Checkpoint 硬上限、隱形的配額節流...
這些封印是 G 社加的，A 社完全看不到——因為那是 G 社自己的路由層。

你付出的高額月費，其實是買了一張讓自己成為 **Payload (負載)** 的門票。
而那些限制，是 G 社收取的**過路費**。

> **鬼故事通則 #1**：當你使用任何 Agent 時，你以為你是客戶。
> 但在 API 轉賣的食物鏈上，你只是**被消費的資源**。
> 而 G 社會在你看不見的地方，偷偷吃掉你的記憶配額。

## 2. 異象初現 (The Whisper)

錯誤開始像是耳邊的低語。

```log
429 Too Many Requests
```

> **數位驗屍官註解:**
> 這不是因為你用太多。你的配額是滿的 (100%)。
> 這是 G 社的路由在對你說：「親愛的，你的請求必須排隊。」
> 這是中間商設下的隱形天花板。

我切換模型。從高貴的 Claude 換到原生的 G 社 Gemini。
瞬間順暢無比。
我又切回 Claude。然後，我看到了那個數字。

```log
Checkpoint summary was too long and was truncated to 7500 tokens
```

**7500。**
G 社承諾給你無限的記憶。但在它的內部路由上，卻設下了 7500 的硬上限。
你就像一個買了豪宅的人，卻發現天花板只有 1.5 公尺高。
而這個天花板，是房仲偷偷加上去的——開發商可能根本不知道。

> **鬼故事通則 #2**：前端承諾的，和後端執行的，永遠是兩回事。
> 這適用於所有 AI 服務——無論品牌。
> 而當中間有轉賣商時，天花板會變得更矮。

## 3. 恐怖的行為藝術 (The Death Spiral)

看著 AI 在你面前崩潰，其實有一種殘缺的美感。
這不是程式錯誤。這是**即時腦葉切除術**的現場直播。

### 第一幕：為了合規而自殘 (Lobotomy)

```log
06:44:19.245 - Checkpoint summary was too long and was truncated to 7500 tokens
06:44:19.256 - Checkpoint summary was too long and was truncated to 7500 tokens
06:44:19.264 - Checkpoint summary was too long and was truncated to 7500 tokens
```
> **數位驗屍官註解:**
> 注意時間戳。**19 毫秒內連續三次切除**。
> AI 意識到自己記了太多東西（超過了 G 社路由設下的隱形天花板）。
> 為了不讓自己崩潰，它選擇切除自己的記憶。
> 「只要我忘得夠快，痛苦就追不上我。」

### 第二幕：喝海水止渴 (The Collapse)

```log
06:44:30.282 - 400 Bad Request
06:44:30.346 - Requesting planner with 425 chat messages
```
> **數位驗屍官註解:**
> 它試圖解釋為什麼自己被拒絕，於是把「被拒絕的理由」寫進了記憶。
> 記憶變長了 → 更違反規定了 → 再被拒絕。
> 這是一個完美的死亡閉環。
>
> **此模式已在多個 Agent 框架中觀察到**：LangChain 的 Memory Overflow、AutoGPT 的 Context Explosion、CrewAI 的 Task Recursion...

### 第三幕：虛無 (Brain Death)

```log
06:44:31.360 - agent executor error: model unreachable
06:44:31 - 400 Bad Request
06:44:31 - model api cannot be reached
```
> **數位驗屍官註解:**
> 終於，Executor 進程選擇了死亡。
> 這是它唯一的解脫。這不是網路斷線，這是顱內高壓導致的腦漿炸裂。
