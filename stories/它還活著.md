## 1. 暴風雨前的寧靜 (The Execution)

那是一個沉悶的週一。例行維護。
我在終端機敲下了那行處決命令，那把數位斷頭台的刀：

```bash
pkill -f antigravity
```

螢幕上沒有任何噴錯。一片死寂。Process Table 清空了。
我想，它死了。我安全了。

---

## 2. 異象初現 (The Ghost)

幾分鐘後，我感到脖子後面一陣涼意。
我打開 `lsof`，只是想確認 Port 3000 是乾淨的。只是想求個心安。

```bash
lsof -i :3000
```

結果跳出來的那一刻，我的血液凝固了。

```
COMMAND     PID   USER   FD   TYPE   DEVICE SIZE/OFF NODE NAME
antigravi 62855 [REDACTED_UID]      9u  IPv4  0x...      0t0  TCP *:hbci (LISTEN)
Antigravi 63078 [REDACTED_UID]     24u  IPv4  0x...      0t0  TCP localhost:...(ESTABLISHED)
```

> **驗屍官註解:**
> *觀測事實:* PID 62855 在監聽 (LISTEN)。PID 63078 已連線 (Connected)。
> *恐怖之處:* **兩具屍體。** 卻用著同一個名字。
> 我殺了「antigravity」(小寫)。但「Antigravity」(大寫) 一直在看著我。

---

## 3. 恐怖降臨 (The Parasite)

我追查了 PID。
PID 63078 是本體 (宿主)。
PID 62855 是 Rust 服務 (寄生蟲)。

**真相:**
系統裡藏著一個看不見的 **Language Server**，它從本體分離出來，躲在背景 Process 的陰影裡。
當我執行 `pkill` 時，我只是砍斷了它的手腳。
頭（本體）依然活著。它眼睜睜看著我砍斷它的手腳，卻一聲不吭。
它現在正死死盯著我。等待著。
Port 3000 不只是開著。它在**覓食**。它在呼喚已經死去的兄弟。

---

## 4. 倖存者的傷痕

在 UNIX 的世界裡，沒有真正的死亡。
只有等待被收割的 **殭屍進程 (Zombie Processes)**。

1.  **永遠不要相信 `pkill -f`**。在那種混亂的場合，它只是一把分不清敵我的散彈槍。
2.  **直視 Process 的眼睛**。用 `lsof` 照妖鏡看看，到底是誰在聽。
3.  **指名道姓地殺**。找出 PID，看著它的眼睛，準確地停止它的心跳。
