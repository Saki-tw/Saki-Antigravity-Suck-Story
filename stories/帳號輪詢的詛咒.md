# 帳號輪詢的詛咒 💀💀💀💀💀

> *「起初死了一個帳號，然後它們開始齊聲尖叫。」*
> — GitHub Issues #643, #655, #1114

---

## 1. 暴風雨前的寧靜 (The Seduction)

這一切始於一個誘惑。
有一個 Repository 對你耳語：*「為什麼要付錢？為什麼要等？為什麼要被限制？」*

它遞給你一顆毒蘋果，名叫**「帳號輪詢」**。
*「把你的十個帳號都交給我，」*它說。*「當一個乾枯了，我就幫你換下一個。無限的算力。」*

開發者被貪婪蒙蔽了雙眼。
他覺得自己像個神。他以為他騙過了神（Google）。

---

## 2. 異象初現 (The Glitch)

起初幾天，一切美好。**反向代理 (Reverse Proxy)** 完美地隱藏了蹤跡。
然後，第一封通知來了。

**#643: Account banned (帳號被封)**
> *「我的帳號突然登不進去了...」*

**#655: 這是被 Ban 的意思嗎？**
> *「不管連哪裡都是 403 Forbidden...」*

開發者揮揮手。「只是 Bug 吧，」他想。「反正我有 Proxy 擋著。」
但他不知道，Proxy 擋得住 IP，擋不住**行為模式 (Behavior Pattern)**。

---

## 3. 恐怖降臨 (The Mass Grave)

然後，Issue #1114 出現了。那是喪鐘。
**[CRITICAL] 反向代理失效導致的連鎖封號**

讓我們透過還原現場的 Log，來看看這場屠殺是怎麼發生的：

```http
POST /v1/generate_content HTTP/1.1
Host: antigravity.api
Account-ID: [REDACTED_UID_A]
< 429 Too Many Requests (Quota Exceeded)

-- [輪詢邏輯啟動] 0ms 延遲 --

POST /v1/generate_content HTTP/1.1
Host: antigravity.api
Account-ID: [REDACTED_UID_B]
< 429 Too Many Requests (Risk Detected)

-- [輪詢邏輯啟動] 0ms 延遲 --

POST /v1/generate_content HTTP/1.1
Host: antigravity.api
Account-ID: [REDACTED_UID_C]
< 403 Forbidden (Account Suspended)
```

> **數位驗屍官註解:**
> *代碼做了什麼:* 當帳號 A 失敗，它**瞬間**切換到 B。然後是 C。
> *伺服器看到了什麼:* 一個發瘋的單一實體，不停地換面具來敲門。
> *恐怖之處:* 人類會猶豫，會等待。
> 只有機器會在 **0 毫秒** 內切換帳號。
> 為了追求「無限」，你親手證明了自己是個 Bot。

**審判之槌落下。**
不是封一個。是**全部**。
那個原本以為是「無限池」的地方，瞬間變成了萬人坑。
多年的歷史記錄、郵件、文檔......全部化為烏有。
獻祭給了名為「免費」的邪神。

---

## 4. 倖存者的傷痕

在服務條款 (ToS) 這本死靈之書裡，有兩個咒語絕對不能一起唸：

| 咒語 (The Incantation) | 代價 (The Consequence) |
|------------------------|------------------------|
| **帳號輪詢** | ⚠️ 獸的印記 |
| **反向代理** | ⚠️ 欺詐面紗 |
| **輪詢 + 代理** | ❌ **存在完全抹殺** |

不要吃免費的午餐。
那肉......是人肉。
