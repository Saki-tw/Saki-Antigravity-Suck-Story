# 帳號輪詢的詛咒 💀💀💀💀💀

> *姑妄言之姑聽之，豆棚瓜架雨如絲*
> *料是厭作人間語，愛聽秋墳鬼唱詩*

> *「然後，他們的帳號都消失了。」*
> *「就像從未存在過一樣。」*

---

## 序：這不只是某個專案的詛咒

這個故事發生在一個聲稱可以「無限使用 AI」的開源專案上。
具體來說——某個 lbj 什麼的開發者創建了一個知名的 Antigravity 帳號管理專案。

它有 **1,300+ 個 Issues**，**68 個 releases**，**49 個 contributors**。
看起來是個成熟的專案。文檔寫得漂亮，星星閃閃發光。
它承諾可以透過 Google Language Server 反向代理，解析 G 社內部的 Claude 呼叫格式。
用 A 社 Claude Code 的介面，白嫖 G 社 Antigravity 內的 Claude 配額。

*「無限的算力，」*它低語著，*「只要你把帳號交給我。」*

但這個詛咒的本體——**帳號輪詢**與**反代理**——存在於無數類似的專案中。

無論是 Antigravity、ChatGPT、Claude、Gemini，還是任何需要付費的 AI 服務，
只要有人試圖用「技術手段」繞過限制，這個詛咒就會降臨。

因為**全知之眼從未閉上**。

它的正則表達式大概是這樣的：
`/rotation|proxy|pool|unlimited/i.test(repo.README)`
只要匹配，你就會被加入觀察名單。

> **真實 Issue 紀錄（來自該專案）：**
> Issue #1357: 「更新4.0.8版本立馬掉了60%額度，從5h變成3d冷卻」
> Issue #1349: 「我的claudecode的opus4.5已經429三天了」
> Issue #1361: 「配額保護失效」
> Issue #1351: 「API Error: 403 - All 3 attempts failed」
> — 這些不是虛構。這些是真實的哭號。

---

## 第一幕：嵐の前の静けさ（暴風雨前的寧靜）

在 GitHub 的某個角落，有一個 repo。

它承諾可以幫你管理多個帳號，自動切換配額。
「太棒了，」開發者們想。
「這樣我就可以無限使用 AI 了。」

他們下載了它。安裝了它。使用了它。

一切看起來都很完美。

就像暴風雨來臨前的寧靜。
那種天氣，黎明時怎麼看都是晴的。
但鳥都不叫了。

> **鬼故事通則**：這種專案存在於每一個 AI 生態系中——
> GPT 有「帳號池」、Claude 有「多重代理」、Gemini 有「配額共享」...
> 它們的核心邏輯都是一樣的：**用量產生風險**。
> 而 A 社在這層面上，完全是另一層面的受害者——它只看到流量暴增，然後被迫執行防禦措施。

## 第二幕：異変の兆し（異變的徵兆）

幾天後，Issue 開始出現。最初只是耳語。

**#643: Account banned**
> *「我的帳號突然無法登入了...」*

**#655: 這是被封號的意思嗎？**
> *「顯示 403 錯誤...到處都是 403...」*

**#646: 新加的兩個帳號似乎被風控了**
> *「看著有滿額，但是用不了...像是被什麼東西盯上了。」*

然後，有人發現了真相——在問題海洋的最深處，一個帶著驚嘆號的紅色標籤：

**#1114: [嚴重] 代理在 503/500/429 錯誤時無退避延遲，可能導致帳號封禁**

> **數位驗屍官註解：**
> 這個 Issue 就像是一封來自深淵的警告信。
> 有人已經掉下去了。有人爬了上來。
> 但大多數人...只是消失了。
>
> **此現象的普遍性**：
> - OpenAI 的「帳號池」專案：多次大規模封禁
> - Claude API 的「共享代理」：Anthropic 的檢測系統
> - Google AI 的「配額繞過」：Google 的全知之眼

## 第三幕：恐怖の降臨（恐怖的降臨）

讓我們重建現場的 Log，看看這場屠殺是如何發生的：

```http
POST /v1/generate_content HTTP/1.1
Host: [any-ai-service].api
Account-ID: [REDACTED_UID_A]
< 429 Too Many Requests (Quota Exceeded)

-- [輪詢邏輯觸發] 0ms 延遲 --

POST /v1/generate_content HTTP/1.1
Host: [any-ai-service].api
Account-ID: [REDACTED_UID_B]
< 429 Too Many Requests (Risk Detected)

-- [輪詢邏輯觸發] 0ms 延遲 --

POST /v1/generate_content HTTP/1.1
Host: [any-ai-service].api
Account-ID: [REDACTED_UID_C]
< 403 Forbidden (Account Suspended)
```

> **數位驗屍官註解：**
> *代碼做了什麼：* 帳號A失敗後，**立即**切換到B。然後是C。
> *伺服器看到了什麼：* 一個瘋狂地更換面具、不停敲門的存在。
> *恐怖的關鍵：* 人類會猶豫。人類會等待。
> **0毫秒**切換帳號的，只有機器。
> 在追求「無限」的過程中，你證明了自己是 Bot。

那些「神奇」的功能——

**帳號輪詢**：在多個帳號之間切換。
**反代理**：繞過官方 API 限制。

它們不是功能。
**它們是詛咒。**

任何 AI 服務商的系統都看到了一切。全知之眼從未閉上。

它看到同一個 IP 在不同帳號之間瘋狂切換。
它看到請求頻率遠超正常人類使用。
它看到了異常行為的模式——Bot 的指紋。

然後——

**審判之槌落下。**

不是一個帳號。
是**所有**帳號。
永久。

多年的郵件、文檔、照片...全部化為灰燼。
作為獻給「免費」這個邪神的祭品。

## 結語：生存者の傷跡（生存者的傷痕）

當你看到這兩個關鍵字同時出現時，無論是哪個 AI 服務，請立刻逃離：

| 關鍵字 | 風險 | 適用範圍 |
|--------|------|----------|
| **帳號輪詢 / Account Rotation** | ⚠️ 獸之刻印 | 所有 AI 服務 |
| **反代理 / Reverse Proxy** | ⚠️ 詐欺的面具 | 所有 API 服務 |
| **免費無限 / Unlimited Free** | ⚠️ 邪神的低語 | 任何付費服務 |
| **帳號輪詢 + 反代理** | ❌ **存在的完全抹消** | 全宇宙 |

因為實作這些功能的人——
他們從來沒有讀過服務條款。
他們從來不在乎你的帳號會發生什麼事。

他們只在乎「能不能跑」。
不在乎「跑完之後你還剩什麼」。
