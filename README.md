# Antigravity Suck Story 🦋

> *「就像一隻自肚腹裏升起的淡藍色蝴蝶...」*
> *我們不是在抱怨。我們是在欣賞這場災難的藝術性。*

這是一個關於 **Antigravity**（那款真實存在的、類似 IDE 的 Agent 應用）的**社會實驗**。
這不只是一個 Bug Tracker，這是一個**數位怪談俱樂部**。

我們是一群「樂子人」。我們花錢買了 Google 的產品，卻發現自己買到的是一張參觀「企業級鬧鬼」的門票。
我們在這裡，不是為了憤怒（那太累了），而是為了**共鳴**。

## 🧪 實驗假設

我們懷疑，這個世界上有成千上萬的人被 Antigravity 衝康過。
他們或許以為是自己太笨，或許以為是網路不好。
但我們知道真相：**這是一場精密的、結構性的、充滿黑色幽默的數位霸凌。**

這裡收集的每一個 Story，都是一隻標本。
記錄了那些「機器產生自我意識來拒絕工作」、「商業合約導致的邏輯精神分裂」以及「前端 UI 對人類進行的煤氣燈效應」。

## 如何加入這場舞會

1.  **Fork 這個專案**。
2.  寫下你的遭遇。越離奇越好，越荒謬越好。
3.  貼上 Log。那是我們在這個荒謬世界中唯一的真實。
4.  發 PR。讓我們一起嘲笑這個世界。
